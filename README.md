# Living Legacy Scoping Application

A streamlined application to help scope and plan personalized life story projects. This tool guides users through defining the scope of their legacy project, ensuring clarity on content, audience, and focus areas before beginning the capture and writing process.

## ğŸ¯ Purpose

The Living Legacy Scoping Application helps users:
- Define the type of legacy project they want to create
- Identify their target audience
- Narrow the scope to manageable, meaningful areas
- Create a clear project roadmap before beginning interviews or writing

## âœ¨ Features

### Legacy Type Selection
Choose from various legacy formats:
- **Full Life Experience** - Comprehensive life story from birth to present
- **Words of Wisdom** - Life lessons and advice for future generations
- **Growing Up** - Childhood and formative years
- **Professional Life: My Career** - Career journey and professional achievements
- **Family Heritage** - Family history and cultural traditions
- **Love & Relationships** - Personal relationships and their impact
- **Overcoming Challenges** - Stories of resilience and perseverance
- **Passions & Hobbies** - Deep dive into personal interests

### Audience Identification
Help users define who the legacy is for:
- Immediate family (children, grandchildren)
- Extended family and future generations
- Professional colleagues and mentees
- General public or community
- Specific individuals

### Smart Scoping Questions
Dynamic follow-up questions based on selections to refine:
- Time periods to cover
- Key themes and topics
- Level of detail desired
- Sensitive topics to address or avoid
- Story format preferences

## ğŸš€ Getting Started

### Prerequisites
- Python 3.8 or higher
- pip (Python package installer)

### Installation

1. Clone the repository:
```bash
git clone https://github.com/grparadise67/living-legacy-scoping.git
cd living-legacy-scoping
```

2. Install required dependencies:
```bash
pip install -r requirements.txt
```

### Running the Application

Start the Streamlit application:
```bash
streamlit run app.py
```

The application will open in your default web browser at `http://localhost:8501`
